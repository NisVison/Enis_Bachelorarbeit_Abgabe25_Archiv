# Bachelorarbeit Hochschule Emden/Leer
## Froschungsfrage: **Untersuchung dreier Verfahren zur Darstellung von Panoramabildern mit visuell optimierter Tiefenwahrnehmung und ihren Grenzen**

Dieses Repository enthält die Materialien, Quellcodes und Ergebnisse zur Bachelorarbeit  

Die Arbeit untersucht drei unterschiedliche technische Ansätze zur Darstellung räumlicher Tiefe in webbasierten 360°-Szenarien auf 2D-Monitoren und vergleicht diese hinsichtlich wahrgenommener Tiefenwirkung, Stabilität und praktischer Eignung für Webanwendungen.

## Abstract

Ziel dieser Arbeit ist die vergleichende Untersuchung dreier Verfahren zur Darstellung räumlicher Tiefe in webbasierten 360°-Umgebungen auf 2D-Monitoren. Im Fokus steht die Frage, inwieweit unterschiedliche technische Ansätze unter den Einschränkungen einer Webumgebung eine überzeugende Tiefenwahrnehmung ermöglichen.

Hierzu wurden drei Szenarien implementiert:  
(1) ein vollständig geometriebasiertes High-Poly-3D-Szenario als Referenz,  
(2) ein bildbasiertes 360°-Panorama mit Depth Map sowie  
(3) eine geometriebasierte Low-Poly-Rekonstruktion mit gebackenen Texturen und gezielten Rendering-Optimierungen.

Die Szenarien wurden in einer explorativen Online-Studie hinsichtlich subjektiver Tiefenwahrnehmung, relativer Distanzurteile und wahrgenommener Stabilität bewertet. Die Ergebnisse zeigen deutliche Unterschiede zwischen den Verfahren: Während das High-Poly-Szenario die höchste Tiefenqualität bietet, ist es für webbasierte Anwendungen nur eingeschränkt geeignet. Bildbasierte Depth-Map-Verfahren sind äußerst effizient, weisen jedoch wahrnehmbare Verzerrungen auf. Die Low-Poly-Rekonstruktion stellt einen praktikablen Kompromiss zwischen visueller Qualität und Performance dar.
